   $ mkdir -p data

   $ python app.py

   $ curl -X POST http://localhost:8000/ask \
     -H "Content-Type: application/json" \
     -d '{"question": "What are the main topics?"}'

📝 CONFIGURATION
════════════════════════════════════════════════════════════════════════════

Environment variables in .env:

HF_TOKEN                          # HuggingFace API token (REQUIRED)
MODEL_ID                          # LLM model (default: Llama-3.1-8B)
EMBEDDING_MODEL                   # Embedding model (default: all-MiniLM-L6-v2)
LLM_MAX_TOKENS                    # Max output length (default: 512)
LLM_TEMPERATURE                   # Creativity (default: 0.7)
SIMILARITY_TOP_K                  # Retrieved documents (default: 3)
DATA_DIR                          # PDF location (default: ./data)
STORAGE_DIR                       # Index location (default: ./storage)
API_PORT                          # Server port (default: 8000)
CORS_ORIGINS                      # CORS policy (default: *)

════════════════════════════════════════════════════════════════════════════
📖 FILE DESCRIPTIONS
════════════════════════════════════════════════════════════════════════════

app.py
  - FastAPI application with lifespan management
  - HTTP endpoints: GET /, GET /health, POST /ask
  - CORS middleware
  - Error handling with proper HTTP status codes
  - Logging configuration
  - Runs as: python app.py

rag_engine.py
  - RAGEngine class (singleton pattern)
  - Document loading from ./data
  - Index building with SemanticSplitterNodeParser
  - Index persistence to ./storage
  - Query processing with source extraction
  - Re-uses index if already built

config.py
  - Loads .env file using python-dotenv
  - Validates required environment variables
  - Type-safe configuration access
  - Path management and validation

models.py
  - AskRequest: Pydantic model for POST /ask input
  - AskResponse: Response with answer + sources
  - SourceMetadata: File name + relevance score
  - ErrorResponse: Error details with code

requirements.txt
  - FastAPI 0.104.1
  - Uvicorn 0.24.0
  - LlamaIndex 0.9.48
  - Torch 2.1.1
  - Transformers 4.36.0
